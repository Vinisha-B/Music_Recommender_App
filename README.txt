# 🎵 Spotify Music Recommendation App

A Streamlit-based web application that recommends songs based on either:

- A user’s favorite track (using content-based filtering)
- A selected mood (like Happy, Sad, or Energetic)

## 🔍 Features
- Content-based song recommendation using cosine similarity
- Mood-based song filtering using audio features (valence, energy)
- Clean web interface with Streamlit

## 🧠 Technologies
- Python, Pandas, Scikit-learn, Streamlit
- Deployed using Streamlit Cloud

## 📁 Dataset
Spotify audio features dataset (from Kaggle or Spotify API)

## 🚀 How to Run
1. Clone the repo
2. Run `streamlit run app.py`
3. Or deploy via [https://streamlit.io/cloud](https://streamlit.io/cloud)
